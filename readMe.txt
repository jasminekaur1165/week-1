Main Class is MainWeek1Project.java

Subject (Observer Pattern) is StickerShopSubject.java which notifies observer classes what kind of sticker to display

Observers (Observer Pattern) are MobileObserver.java and LaptopObserver.java which are used to diplay mobile sticker and laptop sticker respectively

Decorators (Decorator Pattern) are Hat.java, RocketShoes.java and SuperPet.java which are basically accessories used to decorate stickers of type BatMan, IronMan, SuperMan and CaptainAmerica at runtime.
